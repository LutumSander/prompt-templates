# Prompt Templates Library

My version-controlled prompt templates for GTM outreach campaigns.

## Structure
- `/clients` - Client-specific templates (Harmoney, VanRoey, FenekO)
- `/templates/base` - Base templates to build from
- `/shared-variables` - Reusable variables and snippets

## How to Use
Each template uses {{variable_name}} format for easy substitution in Clay.com
